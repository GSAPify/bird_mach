# ADR-0017: Grafana for Dashboards

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on grafana for dashboards.

    ## Decision
    We will implement grafana for dashboards as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
