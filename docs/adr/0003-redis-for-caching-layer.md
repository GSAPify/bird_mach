# ADR-0003: Redis for Caching Layer

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on redis for caching layer.

    ## Decision
    We will implement redis for caching layer as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
