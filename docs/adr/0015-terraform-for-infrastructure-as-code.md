# ADR-0015: Terraform for Infrastructure as Code

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on terraform for infrastructure as code.

    ## Decision
    We will implement terraform for infrastructure as code as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
