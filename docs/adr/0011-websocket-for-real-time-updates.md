# ADR-0011: WebSocket for Real-time Updates

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on websocket for real-time updates.

    ## Decision
    We will implement websocket for real-time updates as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
