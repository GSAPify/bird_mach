# ADR-0023: Feature Flags for Gradual Rollouts

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on feature flags for gradual rollouts.

    ## Decision
    We will implement feature flags for gradual rollouts as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
