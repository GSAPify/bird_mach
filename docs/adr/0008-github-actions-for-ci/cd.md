# ADR-0008: GitHub Actions for CI/CD

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on github actions for ci/cd.

    ## Decision
    We will implement github actions for ci/cd as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
