# ADR-0010: Celery for Background Tasks

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on celery for background tasks.

    ## Decision
    We will implement celery for background tasks as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
