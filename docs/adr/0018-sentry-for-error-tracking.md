# ADR-0018: Sentry for Error Tracking

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on sentry for error tracking.

    ## Decision
    We will implement sentry for error tracking as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
