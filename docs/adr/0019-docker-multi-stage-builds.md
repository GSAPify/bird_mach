# ADR-0019: Docker Multi-stage Builds

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on docker multi-stage builds.

    ## Decision
    We will implement docker multi-stage builds as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
