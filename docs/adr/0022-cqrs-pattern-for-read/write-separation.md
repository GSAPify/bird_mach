# ADR-0022: CQRS Pattern for Read/Write Separation

    ## Status
    Accepted

    ## Context
    As Mach grows into an enterprise platform, we need to decide on cqrs pattern for read/write separation.

    ## Decision
    We will implement cqrs pattern for read/write separation as a core platform capability.

    ## Consequences
    - Improved scalability and maintainability
    - Additional infrastructure requirements
    - Team needs to learn new patterns
